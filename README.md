## サイト概要
- 地域の悩み相談、引越し情報まとめサイト
- 引越しの際に必要な情報や便利な情報を投稿、共有することができ、
地域住民の困りごと知恵袋(相談Q&A)もできるサイト

### テーマを選んだ理由
DMMに参加するために上京して初めて一人暮らしをする中で個人のブログなどでしか情報が手に入らず、引っ越してから知っておいた方が良かったと思う情報を知ることが多かったので、それらをまとめるサイトがあれば漏れなく情報を集められたのではないかと考えたこととランキングなどで表示されていれば、必要な情報を最小限の労力で得られるのではないかと考えたことがきっかけです。地域住民の悩み事の相談ができる機能を追加することで、引越した後でも利用してもらえるようにしました。また、そこの部分で引越し予定の地域の状況を知れたらと考えています。

### ターゲットユーザ
引越しを経験したことのない方や引越しになれていない方
(一人暮らしを始める学生、転勤となった社会人など)

### 主な利用シーン
引越しの前後の情報収集の場合
住居に関する悩みなど相談したい場合

## 機能一覧
###　ユーザー機能
- ニックネーム、メールアドレス、パスワードによる新規登録
-メールアドレス、パスワードによるログイン
- プロフィール画像(非同期プレビュー、 S3へ保存)、自己紹介文登録
- 退会、ログアウト機能
- 詳細ページにタブメニュー(閲覧履歴、ブックマーク一覧、投稿一覧)
- 解決済み比率やベストアンサー比率のグラフ化(chartkick)
- 通知機能

### 投稿機能
- タグ付機能
- 記事、質問の画像埋め込み投稿機能(summernote-rails)
- いいね、共感機能(非同期通信)
- ブックマーク機能(非同期通信)
- 閲覧履歴機能
- 投稿へのコメント、回答機能(非同期通信)
- 投稿編集、削除機能(非同期通信)
- 評価機能(raty.js)
- ベストアンサー機能
- ナイス機能(非同期通信)
- 詳細画面のトラッキング機能(impressionist)

### 検索機能
- タグ一覧表示、タグ検索
- キーワード検索機能(タイトル、本文からあいまい検索)
- 並び替え、絞り込み機能(投稿時間、いいね・共感・ブックマーク数、評価で並び替え、回答の状態で絞り込み)

### その他機能等
- ページネーション機能(kaminari)
- 無限スクロール機能(infinite-scroll.js)
- パンくずリスト機能(gretel)
